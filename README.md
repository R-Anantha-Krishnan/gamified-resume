# Gamified Resume

A React + Vite + Phaser 3 demo app that turns a resume timeline into an auto-running side-scroller.

## Features

- Side-scrolling platformer with auto-run and jump controls
- Phaser 3 game renderer embedded inside React
- Achievement collectibles with popups and progress tracking
- Timeline progress bar at the bottom of the page
- Recruiter-friendly auto-play mode
- Responsive layout for desktop and mobile
- End screen with resume download and social links

## Run locally

```bash
cd gamified-resume
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The game uses placeholder graphical assets generated by Phaser.
- Achievements are defined in `src/data/achievements.ts`.
- The React UI and Phaser game communicate through events in `src/game/GameCanvas.tsx`.
